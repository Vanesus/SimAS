\section{Introducción}

Un caso de uso es una descripción de un conjunto de acciones ejecutadas por el sistema tras la orden de un actor y especifica el comportamiento de un sistema o de una parte del mismo.

Un diagrama de casos de uso muestra un conjunto de casos de uso, con sus relaciones y los actores implicados. Sirve para modelar la vista estática de un sistema y permite visualizar su comportamiento externo; de esta forma, se consigue conocer qué es lo que debe hacer el programa independientemente de cómo se hará.

Los casos de uso del sistema van a ser especificados según los aspectos siguientes:

\begin{itemize}
\item \textbf{Nombre}: nombre completo y codificación del diagrama de casos de uso.

\item \textbf{Actor}: usuario que interviene en el diagrama.

\item \textbf{Descripción}: descripción breve del diagrama en su conjunto.

\item \textbf{Casos de uso}: enumeración y descripción breve de los casos de uso que intervienen en el diagrama.

\item \textbf{Flujo de eventos principal}: enumeración de los pasos que debe seguir el actor para
alcanzar el objetivo del caso de uso.

\item \textbf{Flujo de eventos alternativo}: enumeración de los pasos que debe seguir el actor en el caso de no alcanzar el objetivo del caso de uso.

\end{itemize}

\section{Actores del Sistema}

Como ya se dijo en el capítulo anterior, el sistema solamente posee un tipo de actor final, papel que
sería representado tanto por los \textit{profesores} como por los \textit{alumnos} que utilicen
el software para re\-a\-li\-zar el análisis de gramáticas de contexto libre.

Por tanto, la especificación de los casos de uso que se presenta a con\-ti\-nua\-ción se contempla
únicamente para el actor del sistema, que se denominará \textbf{Usuario}.

				  \item \textbf{CU-2.2.3.2 Construcción colección canónica elementos LR(1)}: 	construir la 							colección canónica de elementos LR(1) necesaria para llevar a cabo el análisis 								ascendente LALR.			
				  
				  \item \textbf{CU-2.3 Buscar gramática}: realiza la búsqueda en el sistema de una gramática 						de contexto libre guardada con anterioridad.

				  \item \textbf{CU-2.6 Analizar cadena}: \textit{realiza} el análisis sintáctico 									correspondiente de una cadena.
