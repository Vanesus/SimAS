
\chapter{Objetivos}

De acuerdo a la identificación real y técnica del problema que se ha realizado en el capítulo anterior, a continuación se expondrán todos los objetivos funcionales que se pretenden alcanzar con el desarrollo de este proyecto.

\section{Objetivo principal}

El principal objetivo es el de construir una herramienta completa que facilite el aprendizaje del Análisis  Sintáctico tanto ascendente como descendente, de forma que permita a todas las personas interesadas en la materia aprender de una manera sencilla, rápida y amena.

El software deberá permitir realizar las siguientes tareas principales:

\begin{enumerate}

\item Editor de gramáticas de contexto libre.

\item Análisis descendente y predictivo.

\item Análisis ascendente:

\begin{itemize}

\item Análisis SLR.

\item Análisis LR-canónico.

\item Análisis LALR.

\end{itemize}

\item Detección y recuperación de errores.

\item Tutorial.

\item Ayuda.

\end{enumerate}

\section{Objetivos específicos}

Además de las tareas principales comentadas anteriormente, la aplicación deberá cumplir los siguientes objetivos secundarios:

\begin{itemize}
  \item Editor de gramáticas:
    \begin{itemize}
      \item Creación de una gramática de contexto libre.
      \item Edición del vocabulario.
      \item Gestión de producciones.
      \item Validación de la gramática.
      \item Carga y almacenamiento de la gramática.
      \item Generación de informes.
    \end{itemize}
  \item Simulador descendente:
    \begin{itemize}
     \item Construcción de la tabla predictiva.
     \item Simulación de la gramática.
     \item Gestión de errores.
     \item Generación de informes.
    \end{itemize}
  \item Simulador ascendente:
  	\begin{itemize}
  	 \item Construcción de la tabla de análisis LR.
  	 \item Simulación de los métodos: SLR, LR-canónico y LALR.
	 \item Gestión de errores.
	 \item Generación de informes.  	 
  	\end{itemize}
  \item Tutorial
    \begin{itemize}
     \item Conceptos teóricos del análisis sintáctico.
     \item Ejemplos.
    \end{itemize}
  \item Ayuda.
    \begin{itemize}
     \item Instalación y desinstalación.
     \item Interfaz y modulos del programa.
     \item Resolución de problemas.
    \end{itemize}

\end{itemize}
